## What is cogmem-kos?

cogmem-kos is a **Knowledge Operating System kernel** that provides:

- **Unified Retrieval** - Text search, vector search, and graph traversal through a single API
- **Entity Graph** - Automatic entity extraction and Wikipedia-style entity pages
- **Agent Subsystem** - Event-driven agents for chunking, embedding, and enrichment
- **Framework Agnostic** - Works with LangChain, CrewAI, AutoGen, or any framework

## Core Concepts

### Items & Passages

**Items** are your source documents (emails, files, notes). They get chunked into **Passages** for search and embedding.

### Entities

Named entities (people, organizations, concepts) are extracted from passages and linked in a knowledge graph.

### Retrieval Plans

Pre-built retrieval strategies that orchestrate multiple search backends:

- **Search First** - Lexical search → hydrate → graph expansion
- **Wikipedia Page** - Entity-centric view with facts and evidence
- **Semantic First** - Vector search → rerank → hydrate
